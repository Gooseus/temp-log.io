Create a Log URL

See below for expression help. Defaults to `({{ time | toLocale:'en-us':'America/New_York' }}): {{ method }} - {{ url }}`

How much log history to store. Defaults to 10 entries.

Documentation

Expression Data
time
Date object for the request timestamp. Should be formatted with available Filters.
method
String for the request method.
url
String for the request URL.
originalUrl
String for the original request URL before parsing.
body
Object with the request body data.
originalBody
String for the original request body before parsing.
headers
Object with the request header data
query
Object with the request querystring data
Expression Filters
stringify
Stringify a JSON object input.
prettyStringify
Stringify a JSON object input prettily with indentation and linebreaks.
toLocale:country:timezone
Use the toLocaleString function of a Date object to create a formatted date string.
i.e. {{ time | toLocale:"en-us":"America/New_York" }}
